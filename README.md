# Study-Buddy-App

A study app to make it easier to learn and retain things more easily.

## 📦 Installation

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Setup

1. Clone the repository:
```bash
git clone https://github.com/9483rud/Study-Buddy-App.git
cd Study-Buddy-App
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application should open at `http://localhost:3000`
