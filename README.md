# Study-Buddy-App

A collaborative study application built with TypeScript and React to help students learn more effectively together.

## 🎯 Overview

Study-Buddy-App is an interactive platform designed to facilitate collaborative learning. Whether you're preparing for exams, working through coursework, or exploring new topics, Study-Buddy-App provides tools to enhance your study sessions.

## ✨ Features

- **Collaborative Study Sessions** - Study with friends and classmates in real-time
- **Interactive Quizzes** - Test your knowledge with custom quizzes
- **Progress Tracking** - Monitor your learning progress over time
- **Study Notes** - Create and organize study materials
- **Q&A Support** - Ask questions and get answers from peers

## 📦 Installation

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Setup

1. Clone the repository:
```bash
git clone https://github.com/9483rud/Study-Buddy-App.git
cd Study-Buddy-App
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application should open at `http://localhost:3000`
